# YouTube 下載器

這是一個簡單易用的 YouTube 下載工具，支援下載 YouTube 視訊為 MP4 格式或音訊為 MP3 格式。

## 功能特色

- 支援 YouTube 視訊預覽
- 可選擇下載為 MP4 或 MP3 格式
- 自訂下載路徑且具有記憶功能
- 同時支援最多 5 個下載佇列
- 顯示下載進度
- 支援繁體中文介面

## 系統需求

- Windows 作業系統
- 不需要安裝 .NET Runtime（已包含在應用程式中）
- 需要網際網路連線

## 使用說明

1. 下載並執行應用程式（免安裝）
2. 複製 YouTube 視訊網址並貼到輸入框
3. 點擊「預覽」按鈕檢視視訊資訊
4. 選擇下載格式（MP4 視訊或 MP3 音訊）
5. 選擇下載路徑（首次使用需設定）
6. 點擊「下載」按鈕開始下載

## 下載設定

- 預設下載路徑：D:\Youtube下載資料夾
- 可透過「選擇路徑」按鈕更改下載位置
- 程式會記住上次使用的下載路徑

## 注意事項

1. 下載限制
   - 最多同時下載 5 個檔案
   - 超過限制時會顯示警告訊息

2. 檔案命名
   - 使用視訊原始標題
   - 自動處理非法字元
   - 自動處理檔案名稱過長的情況

3. 下載狀態
   - 綠色：下載完成
   - 紅色：下載失敗
   - 黑色：下載中

## 錯誤處理

如果遇到以下情況：
- 視訊無法預覽：檢查網址是否正確
- 無法下載：檢查網路連線
- 下載失敗：查看錯誤訊息，可能是視訊已不可用或網路問題

## 開發資訊

- 使用 .NET 8.0 開發
- 使用 WPF 框架
- 使用 YoutubeExplode 函式庫處理下載
